# react-native-template-master-template

Custom **React Native 0.81.5** template.

## Create an app from this template

```bash
npx @react-native-community/cli init MyApp --version 0.81.5 --template github:pincode69/template
```

## Recommended environment

| Tool | Version |
|------|---------|
| Node.js | ≥ 20.19.4 |
| npm | 10.x (with Node 20) |
| JDK | 17 |
| Android compile/target SDK | 36 |
| Android build-tools | 36.0.0 |
| Gradle | 8.14.3 |
